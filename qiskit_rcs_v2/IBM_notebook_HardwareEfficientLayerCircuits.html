Hardware Efficient Layered Circuits¶

In this note book I'll provide a function that created hardware efficient layered circuits like the ones in Phys. Rev. A 104, 022609 (2021). In additon I put some examples on how to both get counts from an IBM backend (you're mileage may vary on this part depending on what machines you have access to) and how to use the built in simulator to generate the ideal frequencies of the output bitstrings.

HE Layered Circuits¶

I'll assume we pick a set of qubits on a line somewhere, and alternate between patterns of cnots like (0,1)(2,3)... and (1,2)(3,4)... If we want to do this on n qubits, we'll just generate a generic n qubit circuit and later transpile it to exactly the qubits on the device we're interested in

In [4]:
def HELayeredCircuit(nqubits,nlayers):
    #we'll output two ciruits, one with and without the final round of measurements.  
    #This makes the simulation step slightly easier
    circ=QuantumCircuit(nqubits,nqubits)
    
    #first layers of single qubits
    for q in range(nqubits):
        random_u = random_unitary(2)
        circ.append(random_u,[q],[])
    for l in range(nlayers):
        ntemp = l%2
        circ.barrier()
        while (ntemp+1)<nqubits:
            circ.cnot(ntemp,ntemp+1)
            ntemp+=2
        circ.barrier()
        for q in range(nqubits):
            random_u = random_unitary(2)
            circ.append(random_u,[q],[])
        
    circ_nomeas = copy.deepcopy(circ)
    for q in range(nqubits):
        circ.measure(q,q)
    return circ,circ_nomeas

Run on Hardware¶

Here's some code to run on hardware and grab some counts. Each job on my machine can run a large number of counts on about 300 unique circuits.

For my 7 qubits I'll pick the line [17,18,21,23,24,25,26]

In [6]:
ncircs = 50
shots = 10000
qubits = [17,18,21,23,24,25,26]
all_circs=[]
all_circsnomeas=[]
for i in range(ncircs):
    circ,circ_nomeas = HELayeredCircuit(7,7)
    all_circs.append(circ)
    all_circsnomeas.append(circ_nomeas)
backend_circs = transpile(all_circs,backend,initial_layout = qubits)

job = backend.run(backend_circs,shots=shots)
print(job.job_id())
job_monitor(job)

    output=np.zeros(2**len(qubits))
    for key in counts(circnum).keys():    
        output[int(key[:],2)]+=counts(circnum)[key] 
    p2 = output/np.sum(output)
    
    
    print('Circuit number {}'.format(circnum))
    print('Variational distance from exp') 
    print(np.sum(np.abs(probs(circnum) - p2))/2)
    print('Variational distance from maximally mixed state')
    print(np.sum(np.abs(probs(circnum) - 1/2**len(qubits)))/2)
    print('Linear cross entropy of exp')
    print(2**len(qubits)*np.dot(p2,probs(circnum)) - 1)
